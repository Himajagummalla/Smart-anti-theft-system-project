# Smart Anti-Theft System

## Overview
The Smart Anti-Theft System is a project aimed at developing an intelligent security system using computer vision and IoT technologies. The system detects unauthorized access and alerts the user in real-time through various communication channels.

## Features
- Motion detection using ultrasonic sensors
- Real-time alerts via SMS and buzzer
- 360 degrees surveillance using servo motors
- Integration with IoT devices for remote monitoring
- processing ide interface for system monitoring and display

## Technologies Used

- Processing ide for display
- Blynk iot for SMS notifications
- Arduino for IoT integration



Usage
Mobile Alerts: Receive SMS alerts when unauthorized access is detected.
